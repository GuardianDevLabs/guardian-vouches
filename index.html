How It Works

1. Add the Bot

Invite Guardian Vouches to your Discord server and give it the necessary permissions.

2. Create a Vouch Channel

Set up a designated channel where members can use the `/vouch` command to leave feedback.

3. Build Your Reputation

Users can now vouch for each other, and the bot will track their reputation score.
